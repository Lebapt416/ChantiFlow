# Configuration PWA (Progressive Web App)

## Icônes requises

Pour que l'installation PWA fonctionne correctement, vous devez ajouter deux icônes dans le dossier `public/` :

1. **icon-192.png** - Icône 192x192 pixels
2. **icon-512.png** - Icône 512x512 pixels

Ces icônes seront utilisées pour l'application installée sur mobile et desktop.

## Génération des icônes

Vous pouvez créer ces icônes à partir de votre logo ChantiFlow en utilisant un outil en ligne comme :
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Ou utilisez un outil de design comme Figma, Photoshop, etc.

## Fonctionnalités PWA

- ✅ Manifest.json configuré
- ✅ Bouton d'installation dans le menu
- ✅ Support mobile et desktop
- ✅ Mode standalone (sans barre d'adresse)
- ✅ Raccourcis vers Dashboard et Chantiers

## Test de l'installation

1. Ouvrez le site sur Chrome/Edge (desktop ou mobile)
2. Le bouton d'installation apparaîtra automatiquement dans le menu
3. Cliquez sur le bouton pour installer l'application
4. L'app s'ouvrira en mode standalone
